myFramework

is a pure CSS framework built in SASS. Its utilizes flex under the covers for the purpose of quickly prototyping websites. The goal of myFramework is to be able to build layouts that:
  • grow to fill the available area
  • scale text to maintain - not yet implemented
  • style common web components like buttons
