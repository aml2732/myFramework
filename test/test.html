myFramework

is a pure CSS framework built in SASS. Its utilizes flex under the covers for the purpose of quickly prototyping websites. The goal of myFramework is to be able to build layouts that:
  • grow to fill the available area
  • scale text to maintain - not yet implemented
  • style common web components like buttons

Borders, Backgrounds, and Text

Use either .border or .border-dotted to specify the border type. Use .border-1 through .border-5 to specify borders between 1 and 5 pixels. Use .border-1-rem to specify a toy case where border width is 1rem; Use the syntax .border-blue-light-2, .border-blue-light-1, .border-blue-neutral, .border-blue-dark-1, or .border-blue-dark-2 to specify the color.

Use .bg-blue-light-3, .bg-blue-light-2, .bg-blue-light-1, .bg-blue-neutral, .bg-blue-dark-1, .bg-blue-dark-2. Make sure your element has a width and height defined or some content.
